# 次バージョン検討メモ

## v2候補: 学習分析モード + リズム加点モード

### リズムモード

- メトロノームまたは簡易ドラム音を鳴らす。
- キー入力が拍タイミングに近いほど加点する。
- ずれ幅を `Perfect / Good / Late / Early` で表示する。
- 通常スコアとは別にリズムボーナスを表示する。

### 平均打鍵時間と評価コメント

- 1キーごとの平均打鍵間隔を表示する。
- 結果画面に短い評価コメントを出す。
- 例: `速さは十分。ミスが少し多めです。`

### 速いキー / 遅いキー分析

- キーごとの平均反応時間を集計する。
- 速いキー ベスト3を表示する。
- 遅いキー ワースト3を表示する。
- 苦手キーを練習候補として提示する。

### 得意ワード / 不得意ワード分析

- 問題ごとの正解率、平均回答時間、ミス回数を保存する。
- 得意ワード、不得意ワードを結果画面に表示する。
- 英単語では弱点ワードだけを再挑戦できるようにする。
