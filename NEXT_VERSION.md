# 次バージョン検討メモ

## v2候補: 学習分析モード + リズム加点モード

2026-06-29 実装開始。

### リズムモード

- メトロノームまたは簡易ドラム音を鳴らす。
- キー入力が拍タイミングに近いほど加点する。
- ずれ幅を `Perfect / Good / Late / Early` で表示する。
- 通常スコアとは別にリズムボーナスを表示する。
- v2初版では `Perfect / Good / ずれ` として実装。

### 平均打鍵時間と評価コメント

- 1キーごとの平均打鍵間隔を表示する。
- 結果画面に短い評価コメントを出す。
- 例: `速さは十分。ミスが少し多めです。`
- v2初版では下部の分析カードへ表示。

### 速いキー / 遅いキー分析

- キーごとの平均反応時間を集計する。
- 速いキー ベスト3を表示する。
- 遅いキー ワースト3を表示する。
- 苦手キーを練習候補として提示する。
- v2初版では速いキー/遅いキーを各3件表示。

### 得意ワード / 不得意ワード分析

- 問題ごとの正解率、平均回答時間、ミス回数を保存する。
- 得意ワード、不得意ワードを結果画面に表示する。
- 英単語では弱点ワードだけを再挑戦できるようにする。
- v2初版ではプレイ中の得意/不得意ワードを表示。

### デモモードと表示調整

- デモ入力で答えを1文字ずつ自動入力する。
- START表示を小さくし、枠からはみ出さないようにする。
- 画像パネルを16:9固定で中央配置する。
- 左コントロール欄はカード内スクロールで下部まで見られるようにする。

### 文字単位スコアと長文入力表示

- 速度評価を1問あたりではなく、1文字あたりの速さとCPM中心にする。
- 高速入力では10000点前後まで伸びるスコア式にする。
- 長い入力でも末尾が見えるように入力追従表示を追加する。
