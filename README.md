# Panel Typing Game

自作問題を読み込んで遊べる、パネルゲーム型タイピングWebアプリです。

## 現状版

- HTML / CSS / JavaScriptのみで動作
- 問題txtフォルダ読み込み、貼り付け読み込み
- `問題,答え` / `問題,サブ` / `音楽` のカンマモード
- 通常モード、タイムアタックモード
- 画像パネルの累積開放、パズルリセット
- ミス時の正解表示
- スコア率ベースのランク判定
- ランキング履歴20件表示
- ミス問題の再挑戦、コピー、保存
- v2ブランチ: リズム加点、平均打鍵時間、キー別/ワード別の学習分析

## 起動

`index.html` をブラウザで開きます。

Chrome / Edge系ブラウザ推奨です。

## 問題txt形式

```txt
タイトル
guitar
keyboard
曲名,アーティスト名
1+1,2
# コメント
bass
```
